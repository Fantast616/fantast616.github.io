字符串压缩问题：

​ 如果在字符串中出现了连续重复子串（重复次数>=2），用“数字+左括号+子串+右括号”方式进行压缩，要求每一次压缩都优先压缩当前（ 子串长度 * 重复次数 ）最长的子串，且优先按重复次数最大压缩，如果最长子串长度一样，优先压缩位置在前的子串，如果压缩后的字符串如果还能压缩，则继续压缩；

  • 输入要求：第一行为字符串s，只包含大小写字符，1<s.length<3000

  • 输出要求：输出压缩后的字符串

  • 时间限制：C/C++ 1000ms，结合输入长度，即要求算法复杂度在O(n^2)

  • 样例1

    • ``` input: aaBcaBcaBcc output: a3(aBc)c

  • 样例3：

    •   input:  aAABBccAABBcc
  output: a2(2(A)2(B)2(c))
